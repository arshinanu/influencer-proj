# Influencer Website — React Template

## Getting Started

```bash
npm install
npm start
```

Opens at http://localhost:3000

## Project Structure

```
src/
  components/
    layout/     → Navbar, Footer
    sections/   → Hero, About, Stats, ContentGrid, Collaborations, Contact
    ui/         → StatCard, PostCard, SocialBadge, Avatar
  pages/        → Home.jsx
  data/         → posts.js (your content)
  styles/       → global.css (theme variables)
```

## Customise

- Replace `Sofia Reyes` with your name in `Navbar.jsx`, `Footer.jsx`, `Hero.jsx`
- Update stats in `Stats.jsx`
- Add real posts to `src/data/posts.js`
- Swap colours in `src/styles/global.css` via CSS variables
